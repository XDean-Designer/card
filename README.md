# card · 剑琅联盟会员卡演示

静态 HTML 演示，通过 GitHub Pages 发布。

## 在线访问

推送并在仓库 Settings → Pages 中启用 **GitHub Actions** 源后，访问：

**https://xdean-designer.github.io/card/**

## 本地预览

直接用浏览器打开 `index.html`，或使用本地静态服务：

```bash
python -m http.server 8080
```
