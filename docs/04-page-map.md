## 2. 前端状态要点

| 场景 | 表现 |
|------|------|
| `validCardCount > 0` | 详情编辑禁用；提示克隆改规则 |
| `shelved` | 列表在「已下架」；不可新办；可重新上架/克隆 |
| Step1 有效期限 | 默认 **永久**；含不限次项目后不可再选永久 |
| Step2 权益开关 | 右侧 Switch；关闭仅收起展开区，**保留已配数据**（E2） |
| Step2 结果卡 | 白底阴影卡；标题=项目范围；副文=次数或折扣；整卡可编辑，右侧删除 |
| 项目范围文案 | 全选在售目录 →「全部项目」；否则全部项目名用 `、` 拼接（可换行，无省略号） |
| 添加权益确定后 | 回 Step2，**对应开关打开且展开区展开** |
| 不限次 × 永久 | 弹窗「不限次项目须先设置卡有效期（不可为永久）」；**取消** / **去设置**；不保留不限次；去设置回 Step1（不预改） |
| 选择项目 | 展示「已选 N 项」；至少 1 项才能确定 |
| Step3 用卡策略 | 退卡：均值 / 减值；业绩：均值 / 原价；各一行公式提示；SVG radio |
| 永久卡 | 退卡/延期列表仍展示「延期」；点击 toast「永久卡无需延期」，不展开面板 |
| 含折扣延期 | 展示费用，可改/0 |
| 退卡 Sheet | 剩余上、消耗下；策略角标均值/减值 |
| 实退改写 | 允许；提交走审计 |
| 步骤条间距 | Step2 步骤条与内容卡间距 **16px** |
