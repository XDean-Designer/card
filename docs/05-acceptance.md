## A. 模板

- [ ] 可创建含面值+项目+折扣的组合卡，并保存 `usagePolicy` 四组合之一（退卡均值/减值 × 业绩均值/原价）  
- [ ] Step2：开关展开配置；结果卡展示全部项目名或「全部项目」；添加权益确定后对应区块保持展开  
- [ ] 列表区分在售 / 已下架；下架后不可新办  
- [ ] 重新上架后可再办  
- [ ] `validCardCount > 0` 时不可改 Step1–3；接口返回锁定错误  
- [ ] 克隆生成新模板，源模板不强制下架  

## B. 办卡

- [ ] 选择既存会员 + 支付成功后生成实例  
- [ ] 实例含策略快照、挂牌价快照、初始 ledger、到期日（非永久）  
- [ ] 面值不因卡到期而清除  

## C. 延期

- [ ] 仅延长 `cardExpireAt`；失效卡可复活  
- [ ] 无折扣免费；有折扣可收费用且流水不计入办卡统计  
- [ ] 永久卡展示延期入口；点击 toast「永久卡无需延期」；接口拒绝  
- [ ] 卡有效期为永久时选「不限次」：弹窗（取消 / 去设置），不保留不限次；「去设置」回 Step1 且不预改有效期  

## D. 退卡

- [ ] 试算：算例 A 均值建议退 **400**、减值 **364**（见 `03-rules-cases.md`）  
- [ ] `suggested = max(0, 实付 − Σ消耗)`，不以剩余封顶  
- [ ] 确认后实例退卡、权益清零；支持改实退并写审计  
- [ ] 无部分退能力  

## E. 非功能 / 边界

- [ ] 缺挂牌价或 ledger 不完整时有明确错误码（可配置拦截）  
- [ ] 本期**不验收**：价目管理、会员管理、补录、业绩运算、预约联调
