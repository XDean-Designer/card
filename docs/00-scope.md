## 本期交付（P0）

- 卡模板：列表（在售/已下架）、三步创建、详情、克隆、上下架  
- 有持卡时模板规则锁定（`validCardCount > 0` 不可改 Step1–3）  
- 权益组合：面值 / 项目 / 折扣  
- 用卡策略：`refundRule` + `performanceRule`（存字段；**业绩不运算**）  
- 办卡、延期、整卡退卡（双模式建议退 + 可改实退 + **审计日志**）  

## 本期明确不做

| 项 | 说明 |
|----|------|
| 项目管理（价目表 CRUD） | 后续另立；卡侧只读挂牌价 |
| 会员管理 | 尚未启动；只消费既存选人接口 |
| 劳动业绩 runtime | 只存 `performanceRule` |
| 自建核销 UI | 消耗来自既存核销或账本 |
| 部分退 | 产品决策：不支持 |
| 适用对象 | 不做 |
| 退卡手续费 / 超额审批 / 备注 | 不做（实退可改须审计） |
| 顾客端预约联调 | 不做 |
